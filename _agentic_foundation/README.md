# Agentic Codebase Template V5 — Orquestração Real

## Includes
- stack profiles
- persona profiles
- planner/executor/auditor/synthesizer pipeline
- task playbooks
- local memory layer
- bootstrap for new projects

## Quick Start
Linux/macOS:
`bash _agentic_foundation/scripts/bootstrap.sh . node-typescript backend performance`

Windows:
`powershell -ExecutionPolicy Bypass -File _agentic_foundation/scripts/bootstrap.ps1 -Root . -Stack node-typescript -Personas backend,performance`

Then update:
- OBJECTIVE.md
- STACK_PROFILE.md
- PERSONA_PROFILE.md
- ORCHESTRATION.md
